# layout-boilerplate

## Installation
```
git clone git@github.com:pavel-trbv/layout-boilerplate.git project_name 
npm install
npm start
```
To build:
```
npm run build
```

## Code linting
```
npm run lint:html
npm run lint:style
```
To run both:
```
npm run lint
```

## Code formatting
```
npm run pretty
```
